# Crust Issues 🍕

Crust Issues is a React-based pizza delivery web app that fetches dynamic product data from a custom Node.js + Express API. The app demonstrates frontend-backend integration with a smooth UI and cart functionality.

---

## 🚀 Features

- 🍕 View a list of available pizzas with names and prices
- 🛒 Add and remove items from a cart (client-side)
- 🔗 Fetches product data dynamically from a backend API
- 🎨 Styled using custom CSS
- 🌍 Live deployment of both frontend and backend

---

## 🛠 Tech Stack

### Frontend
- React
- CSS (custom)
- Netlify (deployment)

### Backend
- Node.js
- Express
- Render (deployment)
